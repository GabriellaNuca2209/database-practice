# Database fun with Spring Data JPA

## Project Description

This application is a simple project that lets users register and recieve email confirmation after registration.

## Functionalities

- **Data Storage**: Users are stored in a database with the help of Spring Data JPA.
- **Email notification**: Users will recieve a confirmation email for registering that will allow them to also see their account details.

## About learning journey

- Validations using Spring Boot Validation
- Exception Handling for Users and Email service
- Email service using SendGrid API
- CRUD operations with Spring Data JPA and PostgreSQL
